Casos de Teste – Autenticação

Esta pasta contém casos de teste relacionados às funcionalidades
de **autenticação de usuários**.

## Funcionalidades testadas
- Login
- Cadastro de usuário

## Padrão dos Casos de Teste
Cada caso de teste contém:
- Objetivo
- Pré-requisitos
- Funcionalidade
- Cenários positivos e negativos
- Escrita baseada em BDD (Gherkin)
